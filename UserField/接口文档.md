## 用户注册部分

- ` "xxx/register/captcha" `

## 发布评论部分
##### 请求URL
- ` "xxx/comments/post" `
  
##### 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|text |是  |string |文本  |
|position{"x","y"} |是  |float64 |发布评论时的经纬度  |
##### 请求示例 

``` 
 {
  "text":"这1是一条评论",
  "position":{
      "x":115.79941,
      "y":28.656973
  }
}
   
```
##### 返回示例 

``` 
  {
    "data": {
        "comment_uid": 1,
        "comment_uuid": "445ac4d24ad8d7878077f0c5d80734d0",
        "message": "成功发布评论"
    }
  }
```
##### 返回参数说明

|参数名|类型|说明|
|:----    |:---|:----- |-----   |
|comment_uid |int |该评论所属建筑的id  |
|comment_uuid |string  |该评论的唯一编号   |
